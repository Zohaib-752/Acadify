# 🎓 Acadify

A modern educational web application built with React, React Router, and Bootstrap.

---

## ✨ Features

- Responsive Design
- Modern UI
- React Router
- Reusable Components
- Portfolio Section
- Courses Section
- Contact Page

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- React Router
- Bootstrap
- HTML5
- CSS3

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Zohaib-752/Acadify.git
```

Go to the project directory

```bash
cd Acadify
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```
